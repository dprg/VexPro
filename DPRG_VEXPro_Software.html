DPRG VEXPro Software

This directory is the top level of the VEXPro software developed by the Dallas Personal Robotics Group and described in the Robot Magazine review of the VEXPro controller in the March/April 2012 edition of the magazine. It contains the watchVideo applications and libVexDuino: an Arduino compatibility library.

Go to watchVideo for the client and server software that captures webcam video on the VEXPro controller, exports it to a PC, and displays it from a java app running on the PC.

Go to libVexDuino for documentation and links to the Arduino compatibility library
